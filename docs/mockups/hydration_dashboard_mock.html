How to read this: the bold white line + soft fill = total tracks (left axis, 0 → 5.5k). The colored lines = % of the corpus hydrated by each source (right axis, 0 → 100%). As the backfill runs, the total line climbs AND the source lines climb together — because every new track gets multi-source hydration from the deep matrix. When a source line diverges upward while total is flat, it means entity resolution just matched a bunch of existing tracks to that source (e.g. the tiktok line jumping between -10h and -6h = when the backfill hit the TikTok endpoints). The dashed radio (airplay) line starts late because its endpoint is gated behind a Chartmetric sub-resource we only just unlocked. The real version will have toggle buttons in the top-right (1h / 24h / 7d / 30d) to zoom the time window, and hover tooltips with exact values per bucket.
